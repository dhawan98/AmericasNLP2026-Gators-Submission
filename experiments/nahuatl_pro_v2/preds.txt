Se siuatsin ika ixonbelo xochikoskatl tlanauak tepetl tlachia.
Se tlakatsin ika itlaken tlayouak uan isonbelo moketski ipan se tetl kanin tepetik, ijtik kuauitlamej.
Se tlatkiistak tlacatl tlapouijtok ipan se kuautlaixpankali ijtik se ueyi kaltlamachtiloyan, uan miakej maseualmej tlakakistokej.
Ome tlakamej motlaliaj ma kinmaixkopinili, se ika ikoton istak tlaxochitlakuioli uan okse ika se payotl xochijyo.
Sekimej tlakamej ika inmaltepetlaken tlejkojtokej itech se kalmimiloli chichiltik.
Eyi siuamej ika inmialtepetsotsol kipixtokej imbasojuan uan mononotsaj, ik uejka oksekimej maseualmej kinchixtokej.
Se ouamilli kiyaualojtok kuaujtlatzacuillotl uan mixtentok.
Se ima kitzkihtok se kuetspaltsin.
Se xochitl kamojtik tlen kipia istak ixiujyo non yotlaxochixtok ipan se xiuitl.
Se kuauxiuitl ika xochikostik.
Se chichilistakxochitl kualtsin ika ixiujo xoxokuik tlen amo kuali nesi.
Se kostik senpoalxochitl itlan se tepantli.
Se ojtli tlatlajko kuajyo uan semi ayauitok.
Se kuaxiloxochitl kostik uan xoxoktik ixijyo.
Se matlalxochitl tlen yolotl kostik.
Se xoxoktik ixtlauak ika kuajyo, uan ik uejka tepeyo ika isepayayo iyak, tsikitsin mixtli kajki.
Nahui tlakaj tlatlakualchijchiuaj itech se kaltsintli. Ome siuamej kitetejkej xouikilitl uan se okichtli kipixtok se apastli.
Se siuatsin ika iuanpil xochiyo tekiti nakatl ipan se kuautlapali, inauak onka se kaxitl ika istak arroz uan se kaxitl xoxoktik.
Sekimej pipiyolpanali ipan se kuajyo.
Tepemej kuajtok uan tlamixtentok uan itlanpa panotokej seki tekonemej.
Se tlakatl nenemi ijtik kuautla ika se mekapali uan tliliuik itlaken.
Se tlapechtli kimantok se tepetl non kiyaualojtok xoxokxiuitl, uan ik uejka ayauitl.
Sekimej maseualmej tlachixtokej kanin kinemakaj tlaxochijkuiloli uan seyok tlamantli.
Ik uejka nesi se altepetsintli ika milajyo, tlamixtentok uan tlakemanian nesi tonatij.
Se kosxochitl non uejkapantik ixochiayo uan iyolo kostik.
Se ueyenakaxochitl non tlaxochistok.
Se atemoxtli temo ipan se tetl kiyaualojtok xiuitl.
Se talkamojtli kemej se yolotl uan okseki ik tlakpak.
Se atoyatl panoua itik seki tenextemej uan xiuitl.
Sekimej tlakaj kitokaj se kuaukaxitl ipan tetokokan uan uejka nesi tepeyo uan kuajyo.

Se xoxokamatlaltik kiyaualojtok kuaujtla uan se kaltsintli tepexitlali.

Se siuatsintli motlakentok ika tlapaltik kamisa uan texotik kueitl moseuijtok itech se ueyi kuauitl.
Ikaj kipixtok ima se kuaxikalkuauitl.
Se akaxochitl tlen kuajyo inauak se tepexitl.
Ipan se tepuscaxitl tliltik motzoyonia chilchiluaki ika comino iaxka iuan okseki tlamantli.
Se milli kiyaualojtok kuajyo uan ik uejka kajki tepeyo, semi tlamixtentok.
Se kuaujkualaktli ika ixiujyo selik mososojtok ipan se ixtlauak xoxoktik, uan ompa uejka nesi se kaltsintli kuaujtlapechtok.
Se coatl tlatlacuilolli nesi xiuhyohtitlan uan itlan se kuauhtepontli tlen moskuitlatok.
Se tepetl kiyaualojtok kuajyo uan onkatej seki kalmej, uan tlachipajtok in iluikak.
Se xoxoktik tlalpipitzin ipan tlali tlatoktok ika kuauxiuitsoli uan xiujtsitsintin. Opochtlampa onka se tlamantli chichiltik.
Se tlacatl nemi itech se ojtli tlen tlaxkaliyo uan neltonatijtok.
Se chichikxochitl non petlani ipan se kuauitl.
Se tokatl tliltik ipan se xiuitl xoxouik.
Se toktli ika kamojtikxochitl uan xoxoktikixiujyo iixpan tlauajki xiuitl.
Se kamojxochitl non yotsintlayouak uan kualtsin ixochiyo.
Se ojtli ipan se tepetsintli inauak kalmej, kiyaualojtok kuajyo uan se tlatentli tlikuauitl.
Se ueyi kuautlatekpantli itlan se tepankali ika kuajyo ikampa.
Se maseuali moseuijtok ikaltempan se kuaukali xochipaltik itlanpa se ueyi kuauitl xoxokuik ixiiyo.
